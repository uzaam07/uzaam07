The Brain Tumor Detection Using Machine Learning project is an intelligent healthcare application designed to assist in the early detection of brain tumors from MRI scan images.

### ⚙️ Technologies

- Python
- TensorFlow
- Keras
- OpenCV
- NumPy
- Pandas
- Matplotlib

## Project Workflow:

```text
📁 MRI Brain Scan Dataset
          │
          ▼

          │
          ▼
📊 Feature Extraction
          │
          ▼
🤖 Machine Learning Model Training
          │
          ▼
📈 Model Evaluation
(Accuracy • Precision • Recall)
          │
          ▼
🧪 Test with New MRI Image
          │
          ▼
🧠 Brain Tumor Prediction
(Tumor / No Tumor)
          │
          ▼
📋 Display Prediction Result
```
